Sans Bot 안내 페이지

음악 재생, GPT 응답, 기본 유틸 명령을 한 번에 사용하는 개인 서버용 디스코드 봇

봇 초대하기 명령어 카테고리

봇 정보

주요 기능

  • 유튜브 단일 영상/재생목록 재생
  • 대기열 관리(스킵, 삭제, 셔플, 반복)
  • 현재 재생 정보 조회
  • GPT 대화 (소유자 전용)

현재 상태

  • 슬래시 명령어 기반
  • 재생목록 최대 50곡 자동 추가
  • 유휴/무인 채널 자동 퇴장
  • 응답 임베드 스타일 통일

명령어 카테고리

상단 카테고리의 명령어를 누르면 전체 명령어 목록 페이지로 이동합니다.

빠른 설치

  1. pip install -r requirements.txt
  2. .env에 TOKEN, GOOGLE_GEMINI_API_KEY 설정
  3. python bot.py 실행

슬래시 명령 즉시 반영이 필요하면 GUILD_ID도 설정하세요.
